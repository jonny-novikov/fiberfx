**Mechanics:** `Workflow` tool, `pipeline(build, verify)` — one builder + one adversarial verifier per hub, each edits its OWN `index.html` (no worktree isolation needed). Hard gates every hero must pass (all nine Apollo gates A+): **degrade** (static SVG in markup; JS only re-renders on click — first paint must not need JS), **motion** (animation wrapped in `@media (prefers-reduced-motion: no-preference)`), svg/voice/storage/links/pager/containers/no-future. Builders must NOT touch the manifest, rebuild cms, or commit. Verifier independently re-runs `cms check` + `cms audit` and judges concept-appropriateness (catches A+ placeholders). **The `degrade` gate checks *visible*-without-JS, NOT *coherent*-without-JS:** a static figure can be self-contradictory and still pass — F6.06 `render.html` shipped hole-0 reading "unchanged @ 41" while the same no-JS frame's payload/code read "changed → 42" (the on-load `render()` masked it), A+ on every gate. The adversarial verifier must confirm the static initial state is internally coherent AND matches the active control's JS output.
