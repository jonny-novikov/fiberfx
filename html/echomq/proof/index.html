The Proof · pillar VI

The system holds,
and it can show you

A bus is only as good as the evidence it can produce about itself.

The Proof is the closing pillar, and it does one thing: it makes the whole system accountable to itself. EchoMQ produces three kinds of evidence — but two are shipped code and one is a named frontier. Conformance (EchoMQ.Conformance) is the bus contract written as runnable scenarios: each drives the public surface against a live server and asserts the externally visible verdict, so the protocol is proven, not asserted — and any runtime that speaks the wire must pass the same set. Telemetry & the read plane (EchoMQ.Meter + EchoMQ.Metrics) is how the running system answers how it is doing without being asked to change — the lifecycle metered the standard :telemetry way at zero cost when absent, and a pure-read plane that observes and never mutates. The benchmark is the named frontier: throughput under load, named here so the picture is whole, never invented so the page stays honest. It is all real code — except the one thing that is not yet on disk, and says so.

Three concerns, one question

The question the Proof answers is simple: can you trust the bus? It answers in three registers — the contract is proven by running it, the running system reports on itself, and performance under load is the frontier named but not yet built. Two are shipped code; the third is greyed, because a depth course names what is coming and builds what is here. Pick a concern to read what it is, the surface beneath it, and whether it ships or waits.

Two concerns are shipped and one is named: the contract is run, the system reports, and the benchmark is honest about being the road not yet built.

The rest of the system

The Proof is the closing reading of the shared substrate. The Queue distributes work over the wire; the Bus broadcasts signals and keeps a replayable log; the Cache serves reads in front of it; the Protocol below them all is the keyspace every key is born to. Proof is the pillar that shows the whole of it holds.
