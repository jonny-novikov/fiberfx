The Protocol · module 04

What holds the
wire together

The keys are fixed, the ids are branded, and a connection that does not match the wire is refused.

The keyspace says where, the record says what, the scripts say how. This module teaches what keeps all three honest over time. Three guards do it: the immutable line — the keys and field names are fixed, so the layer below the line does not change underneath a reader; the branded-id gate — a job id is a 14-byte branded Snowflake under JOB, checked in Lua before a row is written; and the version fence — the reserved {emq}:version key the connector claims or verifies before its first command. All real code, grounded in echo/apps/echo_mq and echo/apps/echo_wire.
