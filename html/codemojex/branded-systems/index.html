Codemojex · chapter C1

The game as branded systems.

Every entity in Codemojex is a 14-character branded snowflake — a three-character uppercase namespace and eleven Base62 characters over ts(41) | node(10) | seq(12) — and the id is the only value that crosses a boundary. It keys the row in Postgres, the entry in Valkey, the job on the bus, and the message that announces a result; because it carries its mint time, it doubles as a cache version and an idempotency token. Fifteen namespaces carry the game today.

Pick a brand below. The tiers it occupies light up — the id is placed only where it is needed.
