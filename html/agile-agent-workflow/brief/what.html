What the chapter covers

An .llms.md brief has five parts, in order: references, requirements, execution topology, agent stories, and the one implementation prompt. The eight modules of A5 teach each part, the practice of running an agent, the thesis, and a workshop that ships a first increment.

Each of the five parts has a role, a module that teaches it, and a real Portal artifact it lands on. The brief is read top to bottom by an agent: references first — links, so the agent reads sources before prose — then the numbered testable requirements, the runtime topology and task DAG, the agent stories with their acceptance gates, and finally the single prompt that runs the whole build in task order. The five parts

An .llms.md brief is one document with five named parts, read top to bottom. References — every source the agent reads, links first, so it reads sources before prose. Requirements — numbered, testable statements, each traced to a user story. Execution topology — the runtime process tree, the build-order task DAG, and the exact files touched. Agent stories — the executable counterpart of a user story, each a Directive the agent runs and an Acceptance gate that closes it. The implementation prompt — the single prompt an agent runs to build the increment in task order, ending on the gates.

Read in that order, the brief assembles a system rather than a pile of snippets: an agent reads the sources, learns the requirements, sees the shape and the order, takes each story to its gate, and runs the one prompt that composes the whole build.

On the Portal

The five parts are not an abstraction; they are the literal sections of the Portal's real briefs. The web bootstrap's f6.1.llms.md opens with References (the Phoenix docs, the Portal facade contract, the design system), then Requirements (F6.1-R1–R8), then an Execution topology (the runtime tree plus the T1–T7 task DAG and the touched files), then Agent stories (F6.1-AS1–AS4), and closes on the implementation prompt. Each A5 module teaches one of these against that real artifact.
