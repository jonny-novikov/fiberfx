A1 · module overview

Why an agile agent workflow

The first module builds the philosophy and the frame. It makes the case for a pragmatic, story-led, agent-driven way of building software, names the two roles and the two layers it runs on, and fixes what “done” means — so every later module is the same loop applied to one more rung. The crucial idea is a pairing: a Claude agent is a fast, tireless implementer of well-specified thin slices, and the human is the source of decomposition, judgement, and acceptance.

By the end of A1 you can say why the workflow works and where it does not, read the framework — the roadmap-and-spec layers, the four artifacts, and the Author/Operator loop — and recognise the one running project, the Portal, that every lesson and the exemplar are built on.

Rationale & intentions

Two habits sink most software built with AI. Prompting without a plan yields code no one can trust; writing one enormous specification yields a document no one finishes. Both skip the unit that makes work tractable — a thin slice of value, proven before the next begins. This module makes the case for that unit and names the framework the rest of the course uses to produce it.

The thesis is a pairing, not a replacement. An agent has throughput and patience a person cannot match, but no taste for what matters; a person has judgement and accountability, but limited tolerance for repetition. Run as a tight loop over thin, provable increments — the human deciding and accepting, the agent specifying and building — the two compound into something neither reaches alone. That is what the modules ahead operate, and what A1 sets out so the operation is deliberate.

What we are building — the agile framework

Before the lessons, three design pieces lay out the framework the whole course runs on — its structure, its vocabulary, and how it moves. Read them as the blueprint; every lesson below puts one piece to work, and every later module applies the same shape to a new rung.

A workflow is a shape you can repeat. These three pieces — the layers, the artifacts, and the loop — are that shape; the lessons put each to work.
