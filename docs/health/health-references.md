## Как использовать

Для каждого модуля дан список из $3$–$6$ опорных источников. При добавлении на страницу блока «Источники»:

1. Возьмите релевантные пункты модуля из этого файла.
2. Сверьте реквизиты с оригиналом (особенно год, том, страницы, $DOI$).
3. Оформите в едином стиле (см. [формат блока](#формат-блока-источники-на-странице)).
4. Для научно-популярных и неэкспертных источников (книги, авторские методики) — пометьте уровень доказательности, чтобы не приравнивать их к рецензируемым исследованиям.

**Иерархия доказательности** (для пометок): мета-анализы и систематические обзоры $>$ $RCT$ $>$ когортные/наблюдательные $>$ руководства и консенсусы $>$ нарративные обзоры и книги $>$ мнение эксперта.

## Формат блока «Источники» на странице

Рекомендуемое оформление блока в дизайн-системе курса (тема главы):

```html
<section class="references">
  <h3 class="ref-title">Источники</h3>
  <ol class="ref-list">
    <li>Автор(ы) (год). <em>Название работы.</em> Источник.</li>
    <li>...</li>
  </ol>
  <p class="ref-note">Образовательный материал. Не заменяет консультацию врача.</p>
</section>
```

Стилевые ориентиры (в духе компонентов курса):

- Блок `.references` — после `.takeaway` и квиза, перед `.nav-prev-next`.
- Рамка/маркеры — акцентного цвета главы (`--{тема}-deep` для бордюра, `--{тема}-bright` для номеров).
- Шрифт списка — `PT Serif`, номера — `JetBrains Mono`; кегль чуть меньше основного текста.
- Ссылки на внешние ресурсы (если есть) — `target="_blank" rel="noopener"`, цвет `--{тема}-bright`.
- $3$–$6$ пунктов на модуль; научно-популярные и авторские источники помечать (например «(книга)», «(авторская методика)»).
