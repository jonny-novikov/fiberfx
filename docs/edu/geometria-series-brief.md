# «Геометрия Шарыгина» — карта серии и рабочая выжимка

> Интерактивный курс **наглядной геометрии для школьников и студентов**. Цель — закрыть пробелы в **пространственном мышлении, культуре построения и визуальной насмотренности**. Хаб серии — страница **Школы Шарыгина** (`/school/sharygin-pokolenia`); все материалы — по маршруту **`/school/{page-name}`**. Держать открытым при работе над любым модулем.

## Концепция и сквозная линия

Геометрия — не свод формул, а **школа видения**: умение представить фигуру в пространстве, повернуть её в уме, увидеть неочевидное построение, прочитать и обосновать чертёж. Курс следует **наглядно-эмпирической концепции И. Ф. Шарыгина**: приоритет наглядных методов и решения задач над формальной аксиоматикой; развитие воображения и «насмотренности». От обзора истории идей — к практике построения, стереометрии, проекциям и конкурсным задачам, с прямым выходом на современные профессии (3D/CAD, архитектура, инженерная графика, дизайн).

Сквозная нить: **видеть → строить → доказывать → применять.** От Евклида и наглядности — к уверенному пространственному мышлению, нужному в вузе и в профессии.

## Аудитория

3. **Чистые статичные SVG-иллюстрации** + **интерактив, где усиливает** (вращение/построение/сечение/развёртка). Интерактивный SVG/HTML (без `<canvas>`), JS-обновляемый. **Готовый приём (M2):** generic `initDrag(svg,onUpdate,constrain)` — точки с `data-drag`, в `<head>` мини-CSS `.geo-svg/.handle/.geo-readout/.ix-reset`, опционально `constrain(handle,x,y)` для привязки к окружности/прямой; показания в `.geo-readout`, кнопка «сброс». Переиспользовать в следующих модулях. **Пошаговое построение (M3):** `initStepper(svg,{cap,count,prev,next,reset,captions[]})` — элементы с `data-step="1..N"` (базовые без атрибута видны всегда), CSS `.geo-svg [data-step]{opacity:0;transition}` + `.step-ctrl/.step-cap`; «назад/далее/сброс», подпись на каждом шаге. **3D-вращение (M4):** вершины 3D → матрицы поворота вокруг X/Y по dx/dy перетаскивания → орто/изо-проекция → обновление рёбер-`<line>`/граней-`<polygon>`; обязателен прозрачный `<rect pointer-events="all"/>` на весь SVG (иначе клик по пустому центру не ловится). **Фолдинг развёртки (M4):** слайдер 0→90°, каждая грань — функция fold(x,y,φ)→[x,y,z] вокруг своей петли, изо-проекция. **Универсальное тело (M6):** `makeSolid(svg,cx,cy,scale)` с `setSolid(rawV)` — нормирует вершины, **рёбра определяет автоматически по минимальному расстоянию** (допуск ~6%), создаёт `<line>` динамически (createElementNS), depth-opacity; работает для тетра/куб/окта/додека/икоса/кубооктаэдра; для селектора — `setSolid` по клику. ⚠️ JS вписывать прямо в кавычный heredoc, не через shell-переменные (`read -d` ломается в sh → пустые переменные → `makeSolid` undefined). **Сечение-полигон (M7):** для тела (V,E) и плоскости {n,D} — пересечение каждого ребра (знак n·v−D меняется → t=fa/(fa−fb)), затем **упорядочивание точек по atan2 в базисе плоскости** (u ⟂ n, w=n×u); рисуется `<polygon>`, вращается вместе с телом; слайдер двигает D (морфинг формы). ⚠️ KaTeX: `$…$` (не `\$…\$` — экранированный доллар не рендерится), кириллицу в формулах оборачивать в `\text{}`. **Запертые карточки:** `.card.locked` (dashed, opacity) + `.lock-note` с inline-SVG замком, текст «доступно по окончании курса» вместо ссылки. **Эпюр Монжа (M8):** foldH(x,y,θ)=[x,y·cosθ,−y·sinθ] поворачивает горизонтальную плоскость вокруг оси x; при 90° фронтальная a′ и горизонтальная a″ проекции точки оказываются на одной вертикали (линии связи); проецирующие лучи и точка гаснут (opacity 1→0). **Перетаскиваемая точка на эпюре:** две связанные точки (a′ выше оси, a″ ниже), drag через nearest-circle; чтение x (вдоль оси), z (высота a′), y (глубина a″). ⚠️ при тесте вращаемых/перетаскиваемых элементов на длинной странице элемент может оказаться НИЖЕ вьюпорта (real-mouse промахивается) — проверять синтетическими событиями (getScreenCTM forward → dispatch MouseEvent). **Отражение/кратчайший путь (M9):** B′=отражение B, |PB|=|PB′|, путь A→P→B = A→P→B′, минимум = |AB′| при P на прямой AB′; ## Опора по истории геометрии (для M1 и углублений)

- **Практические корни:** «гео-метрия» = землемерие; Египет/Вавилон (восстановление границ после разливов Нила).
- **Евклид, «Начала» (~300 до н.э.):** аксиоматическое построение — определения, постулаты, доказательства; образец строгости на 2000 лет.
- **Пятый постулат → неевклидова геометрия:** многовековые попытки доказать постулат о параллельных; **Н. И. Лобачевский (1829), Я. Бойяи, К. Ф. Гаусс** — независимо открыли неевклидову геометрию; пространство может быть искривлено.
- **Р. Декарт (1637):** аналитическая/координатная геометрия — точки как координаты, кривые как уравнения; союз алгебры и геометрии.
- **Проективная** (перспектива Возрождения → Дезарг, Понселе); **начертательная** (Г. Монж, ~1799); **риманова** геометрия кривого пространства (Б. Риман, 1854) → ОТО Эйнштейна.
- **Обучение:** маятник от наглядности к формализму и обратно; ответ Шарыгина — наглядность + решение задач + развитие воображения; в эпоху 3D/CAD «насмотренность» снова критична.
