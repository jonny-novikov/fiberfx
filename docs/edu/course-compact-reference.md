### KaTeX правило (КРИТИЧНО)

**НИКОГДА** не помещать внутри `$...$` или `$$...$$` следующие символы:
`№ ₽ « » " " „ …` или Cyrillic-текст в `\text{}`.

Вместо `\text{₽}` использовать ₽ как plain HTML вне math.
Вместо `\text{реал}` использовать `\text{real}` или `_{real}`.

В JS template literals: `\\%` не `\%`, иначе KaTeX warning.

Опции рендера должны включать:
```js
renderMathInElement(el, {
  delimiters: [{left:'$$', right:'$$', display:true}, {left:'$', right:'$', display:false}],
  throwOnError: false,
  strict: false
});
```

### Спецификация центрального проекта (6.5)

**Калькулятор с 6 ползунками:**
- Текущий возраст (18-65, default 30)
- Целевой возраст (40-75, default 60)
- Доход в месяц (30k-2M, default 150k)
- Ставка сбережений (0-50%, default 15%)
- Текущий NW (0-100M, default 500k)
- Реальная доходность (0-8%, default 3%)

**6 выходных метрик:**
- Горизонт (years)
- Внесено за период (annual savings × years)
- Прогноз капитала к цели (FV в сегодняшних ₽)
- Устойчивый месячный доход (capital × 0.04 / 12)
- Замещение текущего дохода (%)
- Требуемая ставка для 100% замещения

**Дефолтный сценарий (верифицирован):**
- Annual savings: 270k
- FV(NW) = 500k × 1.03^30 = 1,213,631
