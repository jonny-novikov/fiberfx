## A+ Apollo gates

- KaTeX strict: единицы и типографские символы — вне `$...$`; в формулах только команды (`\sum`, `\frac`, `\text`, `\cdot`) и латиница/числа. Скан `\$[^$\n]+?\$` по всем страницам и докам — чисто.
- Результаты интерактивов формируются без `$`-разметки, поэтому KaTeX auto-render их не затрагивает, а преполёт не ложно срабатывает.
- 0 изображений (бюджет валидатора 0; только DOM/computedStyle).
- Ссылки root-relative (`/physics/…`); хост не захардкожен. Вложенных `<a>` нет.
- Каждый модуль завершается квизом; ключи localStorage по схеме `phys-<глава>-<модуль>-quiz`.
- На каждой странице: полоса прогресса, section-nav (scroll-spy), карточки пред./след., «↑ Наверх», хлебные крошки, safety-banner.
