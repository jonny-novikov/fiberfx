# ⚡ Toolkit курса «Электричество и устройства: школьная физика дома»

Самодостаточный toolkit для сборки курса на маршруте **`/physics`** в дизайн-системе **jonnify luxury editorial** (электрическая редакция). По аналогии с toolkit курса «Право».

> **Тезис:** закон Ома, мощность и энергия из школьной физики — аппарат, чтобы безопасно обращаться с током и техникой, не переплачивать за свет и понимать, что происходит в розетке.

## Что внутри
- **Генераторы (в корне):** дизайн-система (`style.py`) и генераторы страниц (`build_home.py`, `build_chapter_landing.py`, `build_module.py`), проверка формул (`verify_physics.py`) и преполёт (`preflight.py`).
- **`validator/`** — headless-валидатор (Playwright, 0 изображений) + сьюты.
- **`site/physics/`** — **собранная Главная страница курса** (`/physics`) + образцы лендинга главы и модуля с интерактивом.
- **`docs/`** — спецификация с директивами, оглавление, статус, источники, build-playbook.

## Быстрый старт
```bash
python3 verify_physics.py          # проверить формулы (21/21)
python3 build_home.py              # собрать Главную -> site/physics/index.html
python3 build_chapter_landing.py   # образец лендинга главы -> site/physics/tok/index.html
python3 build_module.py            # образец модуля (калькулятор + квиз)
python3 preflight.py site/physics/index.html

cd validator
BASE_URL="file:///$(cd ../site && pwd)" node suite.home.js     # 24/24
BASE_URL="file:///$(cd ../site && pwd)" node suite.sample.js   # 18/18
```
Откройте `site/physics/index.html` в браузере.

## Ключевые принципы
- Улучшенная навигация: **полоса прогресса чтения** + липкий **scroll-spy**.
- **Электрическая палитра**, гармонизированная с темами (синий ток, янтарь энергия, зелёный счёт, красный безопасность, фиолетовый устройства, бирюза финал).
- **Моноширинные формулы** (JetBrains Mono) — «приборный» вид.
- Каждый модуль — доступное объяснение + **один интерактив** (калькулятор/визуализация/чек-лист) + **квиз**.
- **KaTeX strict**, **0 изображений**, ссылки **root-relative**.
