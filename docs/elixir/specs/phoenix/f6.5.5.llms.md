# F6.5.5 · agent brief (llms)

> Implementation brief for an agent applying the F0 design system to the Portal. References, traced requirements, the
> execution topology, and a paste-ready prompt. Pairs with the spec [`f6.5.5.md`](f6.5.5.md) and the stories
> [`f6.5.5.stories.md`](f6.5.5.stories.md). The build introduces a root layout (the F0 envelope), a committed `app.css`
> served by the EXISTING `/assets` mount (no bundler), and restyled LOCAL components — all over the facade, no
> `CoreComponents`. Framing for everything emitted: no gendered pronouns for agents; no perceptual/interior-state verbs for
> tools or agents; no first-person narration.

## Execution topology

```text
Runtime render paths (after the binding edit):

   browser HTTP request ─▶ Endpoint plug stack ─▶ Plug.Static (/assets ─▶ app.css)   # static, verbatim
                                                └▶ Router ─▶ controller ─▶ render(conn, :index)
                                                                           └▶ course_html.ex ─▶ wraps in PortalWeb.Layouts :root
   browser HTTP request (LiveView) ─▶ Router ─▶ CatalogLive.mount/3 (disconnected) ─▶ render/1
                                                                           └▶ dead render wraps in PortalWeb.Layouts :root
   socket connect ─▶ CatalogLive.mount/3 (connected) ─▶ diffs/patches the body INSIDE the already-sent :root envelope

   root.html.heex  =  <!doctype><html><head … <link app.css> …></head><body> .route-tag · {@inner_content} · .site-foot </body></html>
   app.css         =  :root{ f0.md-D1 tokens }  +  base CSS  +  .mod/.fig/.pill/.btn/.route-tag/.site-foot   (every clamp() SPACED)
```

```text
Task DAG:
  T1 app.css (tokens byte-for-byte + base + anatomy classes; clamp-spaced)
       └─▶ T2 Layouts module + root.html.heex (envelope links ~p"/assets/app.css"; .route-tag + .site-foot)
            └─▶ T3 binding (portal_web.ex: layouts:[html: Layouts]; :live_view layout:{Layouts,:root})
                 └─▶ T4 restyle course_card→.mod + badge→.pill.live
                      └─▶ T5 restyle panel→.fig, input field, buttons→.btn/.btn.ghost
                           └─▶ T6 verify (compile, suite, clamp-parse, token diff, parity e2e, liveness curl)
```

- **new** — `apps/portal_web/lib/portal_web/components/layouts/root.html.heex` (the F0 envelope).
- **new** — `apps/portal_web/priv/static/assets/app.css` (tokens + base + anatomy classes; replaces the `.keep` placeholder

- **edit** — `apps/portal_web/lib/portal_web.ex` (`:30` `layouts: []` → `layouts: [html: PortalWeb.Layouts]`; `:52`
  `:live_view` macro gains `layout: {PortalWeb.Layouts, :root}`).
- **edit** — `apps/portal_web/lib/portal_web/components/catalog_components.ex` (restyle `course_card`/`panel`/`input`;
  arities/attrs/`data-*` hooks UNCHANGED).
