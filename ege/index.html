Две части — две стратегии

ЕГЭ по профильной математике состоит из двух частей. Первая — короткие ответы по 1 баллу, вторая — развёрнутые решения с обоснованиями по 2–4 балла. Стратегия подготовки в них принципиально разная.

Как мы учим

Принципы подхода

Курс выстроен в стиле академической геометрии: каждая теорема — с доказательством, каждый приём — с обоснованием, каждая задача — в контексте семьи похожих.

I

От типа к задаче

Любая задача ЕГЭ сводится к одному из канонических сценариев. Сначала учимся распознавать тип, потом применяем готовый алгоритм. Никакой «магии» — только узнавание шаблонов.

II

Три метода вместо одного

Каждая задача разбирается тремя независимыми способами: геометрическим, координатным и через «трюк». Это даёт страховку на экзамене — если один метод не пошёл, переключаешься на другой.

III

Видимая математика

Стереометрические задачи разобраны с интерактивной 3D-визуализацией. Можно покрутить чертёж, увидеть проекции, прокликать шаги решения. Понимание глубже текстового разбора.
